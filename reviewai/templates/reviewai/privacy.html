Overview

ReviewAI is a Chrome extension that analyzes product reviews on supported e-commerce websites (such as Amazon, Shopee, Lazada, Walmart, eBay, and others) to detect potentially fake reviews using AI. This Privacy Policy explains how we handle data when you use our service.

2

Data Collection

What We DON'T Collect:

  • Personally identifiable information (names, email addresses, passwords)
  • Financial data or payment information
  • Health, location, or web history data

What We DO Access:

  • The extension reads only public product reviews displayed on the websites you visit
  • Any review data accessed is used solely for AI-powered analysis
  • Review data is not stored or shared with third parties
3

Remote Code

ReviewAI may fetch AI analysis scripts and model data from our secure server at https://www.reviewai.live to perform real-time review analysis.

Important: No personal data is transmitted to our server; only public review content is processed for analysis purposes.

4

Data Use

Data accessed is used exclusively to provide insights on the authenticity of product reviews. ReviewAI operates with the following principles:

No Selling

We never sell user data

No Transferring

We never transfer data

No Sharing

We never share with third parties

Privacy First

Your privacy is our priority

5

Security

Our Security Measures:

  • HTTPS Encryption: All communication with our server uses HTTPS for secure data transmission
  • No Storage: No user credentials, financial info, or personal identifiers are stored
  • No Transmission: Personal data is never transmitted to our servers
  • Public Data Only: We only process publicly available review content
